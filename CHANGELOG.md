## [1.0.0] - 2026-01-29

### Added
- **Ability Creator Editor Window**: Create abilities from `Tools → Ability Creator`
- **Automatic Class Generation**: Generate AbilityData and Ability logic classes
- **ScriptableObject Creation**: Automatically create ability assets with assigned AbilityName
- **AbilityManager Runtime System**: Centralized ability management with pooling support
- **BaseAbility<T>**: Generic base class for implementing ability logic
- **AbilityData**: ScriptableObject base class for storing ability data
- **Pooling System**: Efficient Spawn/Release mechanism for ability instances
- **Batch Operations**: ExecuteAll and CancelAll methods for managing multiple ability instances
- **Edit Mode Tests**: Tests for class generation and asset creation
- **Play Mode Tests**: Tests for runtime execution, pooling, and cancellation logic
- **Debug Logging**: Built-in logging for ability lifecycle (Initialize, Execute, Cancel)
